ACM Programming Contest @ CPP

Where the next top coder at CPP starts

Team Registration


  1. Register an account at: UVa Online Judge
  2. Register an account at: ACM-ICPC Live Archive (You need to link the UVa Online Judge ID)
  3. Login at: ACM-ICPC Live Archive

Rules


  1. NO INTERNET! (Except the Submission System and Java Doc)
  2. Languages: C, C++, Java, and Python 3
  3. Submitted source files must be no larger than 300000 bytes
  4. During the Contest, team members are permitted to communicate with their own team mates or contest officials only
  5. Contest Duration: 3 Hours
  6. For the program submission, please see this page as a reference

Problem Submission


  1. Use "Quick Submit" at: ACM-ICPC Live Archive (Make sure you choose the correct Problem ID
  2. You can check the status of the submission at "My Submissions"
